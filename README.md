# Python Data Analysis Practice

This repository contains my practice exercises and examples for:

- NumPy
- Pandas
- Matplotlib

The exercises are based on practice problems and examples from GeeksforGeeks. The purpose of this repository is to improve my understanding of Python libraries used for data analysis and visualization.

### NumPy
- Array creation

- Mathematical operations
- Reshaping arrays
- Statistical functions
- Linear algebra basics

### Pandas
- Series and DataFrames
- Reading and writing data
- Data selection and filtering
- Handling missing values
- Grouping and aggregation
- Data manipulation

### Matplotlib
- Line plots
- Bar charts
- Scatter plots
- Histograms
- Pie charts
- Customizing plots
